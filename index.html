The Shibe World

STATUS: ALPHA



This game is under an unstable phase, we dont recomend using it as a personal Dogecoin bank.

What is it?

The shibe world is a mmo rpg based on The Mana World that accepts dogecoin transactions in-game. The Shibe World is licensed under the GNU GPL license so you are free to play, share, see the source code and modify it as long as it preserves the same license. It is currently under a playable unstable alpha test. Click here to see who is playing right now.
